# Abhishek Muralidharan - Portfolio

Personal portfolio website built with **React + Vite**. It presents Abhishek's applied AI, backend engineering, projects, skills, achievements, and contact links in a restrained, recruiter-readable interface.

## Sections

- **Hero** - Positioning, photo, social links, resume, and project CTA
- **About** - AI/backend focus, education, leadership, and proof points
- **Skills** - 6-category toolkit across languages, AI systems, backend, data, cloud, and tools
- **Projects** - Selected GitHub work: KJU, MediMax, RAG-implementation, and Market-Researcher
- **Achievements** - Hackathons, ideathons, competitive wins, and leadership roles
- **Contact** - Email, GitHub, LinkedIn, and LeetCode links

## Tech Stack

- React 19
- Vite
- Plain CSS (mobile-first responsive, custom properties)
- Local font packages (Geist, Inter, and JetBrains Mono)

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`.
